Common issues:

Baldur's Gate 1/2 won't start [Linux]

Download "libcrypto.so.1.0.0" & "libssl.so.1.0.0" and put them in a folder somwhere, then link to the folder in "start.sh" by adding the following line:
export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:"/path/to/folder"

Daggerfall falling through geometry [All]

Physics in the DOS version of Daggerfall breaks at high cycle speeds, therefor do one of the following:
1: In DOSBox, enter the command cycles=50000.
2: Install Daggerfall Unity.

Gothic 2 "Access Violation" [All]

Find the file "Gothic II/system/Gothic.ini" and change "musicEnabled" to 0.

The Sims 3 GPU at 100% [All]

Force cap the framerate to 60 FPS using a tool like MangoHUD or NVIDIA Control Panel.
